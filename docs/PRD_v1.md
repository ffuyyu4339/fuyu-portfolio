# PRD v1

## Product Goal

构建付钰的个人作品集网站，展示软件工程背景、Android 应用方向、从服务行业回归技术行业的职业转型、AI 应用开发和数据分析方向的项目实践能力。

## Target Audience

- 上海互联网、AI 应用、数据分析、产品开发相关岗位招聘方
- 技术同伴与开源协作者
- 未来项目合作方

## MVP Pages

- `/`
- `/projects`
- `/projects/[slug]`
- `/workflow`
- `/learning`
- `/resume`
- `/about`
- `/contact`

## Content Source

MVP 使用本地 TypeScript 数据文件，后续可迁移到 MDX 或 CMS。
