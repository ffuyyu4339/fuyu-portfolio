# Constraints

## Technical Constraints

- 使用 Next.js、React、TypeScript、Tailwind CSS、Framer Motion、Lucide React。
- 不引入数据库。
- 不引入复杂后端。
- 不引入重型 CMS 或重型 UI 库。
- 内容优先放在 `src/data`。

## Content Constraints

- 不夸大经历。
- 不虚构工作经验。
- 不使用“精通”等绝对化表述。

## UX Constraints

- 移动端必须可用。
- 所有新增页面必须具备基础响应式布局。
- 视觉风格应理性、工程化、清晰、轻微未来感、不浮夸。
