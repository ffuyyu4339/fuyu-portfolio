MVP RELEASE

个人简约记账本 APP

一款基于 Android 原生开发的轻量级个人财务管理工具。采用 MVVM 架构与 Material Design 设计规范，专注于提供低成本、清晰、可复盘的快速记账体验。

Android Java MVVM SQLite

Project Meta

  • Language Java
  • Architecture MVVM
  • Database SQLite / xUtils
  • Role Solo Developer

问题背景

市面上应用臃肿，用户需要低成本、清晰、可复盘的个人记账需求。核心诉求在于快速记录与精准的财务回顾。

产品目标

打造一款个人日常财务管理 MVP。剥离非核心功能，聚焦输入效率与本地数据的安全持久化。

schema 数据模型

核心业务实体的关系设计，确保高效查询与统计：

  • User Entity 管理用户基本信息，为多用户或未来云端同步做准备。
  • Transaction Entity 核心流水表，记录金额、时间戳、类型（收入/支出）、备注与关联的分类ID。
  • Category Entity 分类字典表，支持自定义图标资源映射与层级关系。

SQLite / xUtils

基于 Java 与 XML，采用标准 MVVM 架构分离视图与逻辑。底层数据库使用 SQLite 结合 xUtils 框架进行轻量级 ORM 操作。遵循 Material Design 规范，保证了代码的可维护性与模块化。

storage 本地化存储

采用 SQLite 作为核心存储方案，实现数据本地优先，无网环境下依然可用。

  • 引入 xUtils 作为 ORM 框架，大幅简化了数据库的增删改查操作，避免手写冗长易错的 SQL 语句。
  • 设计合理的数据库索引，提升按月、按分类进行流水聚合查询的性能。
  • 利用 Android SharedPreferences 存储轻量级用户配置，如预算阈值与 UI 偏好。

pie_chart 可视化统计分析

集成 MPAndroidChart 强大的图表库，将枯燥的流水数据转化为直观的财务健康报告。

brush UI 设计规范

严格遵循 Material Design 规范，打造干净清爽的界面体验：

  • 使用大面积留白与清晰的层级投影（Elevation）区分内容卡片。
  • 定制分类图标与颜色体系，使得财务报表不仅数据准确，而且视觉愉悦。
  • 底部导航栏（Bottom Navigation）与悬浮操作按钮（FAB）的经典结合，确保核心功能一触即达。

stars 项目亮点

  • 移动端业务闭环：从数据采集到统计展现的完整生命周期。
  • 本地优先：无网环境下亦可提供完整流畅的服务，保障隐私。
  • 数据复盘：优秀的趋势图与数据结构支持个人财务健康分析。
  • Android 基础扎实：深入应用四大组件、自定义 View 及本地存储机制。
